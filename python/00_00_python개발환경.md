# 파이썬 기초

## 파이썬 개발환경

### 파이썬이란?

- 배우기 쉬움
- C나 자바보다 간결하게 작성 가능
- 다양한 운영체제에서 실행 가능
- 컴파일 과정 없이 바로 실행 가능
- 객체로 구현



### 파이썬 개발환경 종류

- 대화형 환경
  - 파이썬 기본 인터프리터 : IDLE
  - **주피터 노트북 (실습)** : 웹 브라우저 환경에서 코드 작성
- 스크립트 실행 (금요일 / 평가)



### Python 스크립트 실행

- IDE(ex.Pycharm; 알고리즘), Text editior(ex. Visual Studio Code; 코딩, 웹개발) 등에서 작성한 파이썬 스크립트 파일을 직접 실행
