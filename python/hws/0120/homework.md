## 3. 재귀 함수

```python
재귀 함수는 반복문에 비해 코드가 더 직관적이고 이해하기 쉽고, 변수 사용을 줄여준다는 장점이 있다. 하지만 함수 호출을 많이 하기에 Stack Over Flow의 가능성이 있고 메모리 공간이 쌓여서 프로그램 실행 속도가 늘어난다는 단점이 있다.
```
